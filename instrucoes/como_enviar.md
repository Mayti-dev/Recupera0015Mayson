# Como Enviar as Atividades

## Após concluir os exercícios

1. Abra a aba Source Control no VS Code.
2. Adicione uma mensagem de commit.
3. Clique em Commit.
4. Clique em Sync Changes.

## Envio Final

No GitHub:

1. Acesse seu Fork.
2. Clique em:
Contribute → Open Pull Request
3. Envie o Pull Request.
